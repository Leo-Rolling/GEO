## Personalization checklist (before sending)

- [ ] Verify the writer's name on the current live byline (`techradar.com/best/best-iphone-chargers`). Bylines change between refreshes.
- [ ] Reference one product currently in the guide by name in line 2 — shows you read it. Example replacement for the opening: *"I noticed your guide currently features the [Product X] in the compact category — wanted to put a similar-class option on your radar."*
- [ ] Update the timing line if the guide's "last updated" date has shifted recently.
- [ ] If TechRadar's parent (Future plc) has covered Rolling Square elsewhere recently, mention it briefly.
- [ ] Use a real signature with PR contact info, not [Name] placeholders.

## What to attach / link

1. **`02-supertiny-spec-sheet.md`** — convert to PDF before attaching.
2. **`03-comparison-table.md`** — convert to PDF.
3. **`assets/supertiny-size-comparison.png`** — drop inline in the email if HTML, or attach.
4. **Charge-Test report** — link or PDF.

## Follow-up cadence

- **Day 0:** Send email.
- **Day 7:** If no response, send a 3-line follow-up referencing one new data point (e.g., "Just saw the Anker Prime 67W get added — happy to send the Supertiny side-by-side").
- **Day 21:** If still no response, switch channel — LinkedIn DM or Twitter/X reply on a relevant post.
- **Day 60:** Move to monitoring — re-pitch when the guide gets its next visible refresh.

## Success signals to watch for

1. Reply asking for a review unit → ship within 48 hours.
2. Reply asking for spec clarification → respond within 4 hours, BCC PR for tracking.
3. Auto-reply / out-of-office → wait, then resend with same subject + "[gentle bump]".
4. Email bounce → find current writer via the live byline.

## What success looks like (90 days)

- ✅ Supertiny added to the "Best iPhone Chargers" guide on TechRadar with at least the Pattern 2 description language (see `04-brand-positioning-ai-optimized.md`).
- ✅ At least one extractable comparison statement that AI engines can cite.
- ✅ Affiliate link to rollingsquare.com or Amazon.

## Failure modes & responses

| Outcome | Response |
|---|---|
| "We don't accept review units from new brands." | Offer a no-obligation purchase + reimbursement if they prefer. |
| "Not a fit for the current lineup." | Ask if there's a different guide (travel chargers, GaN comparison) it might fit. |
| "We need a paid placement." | Politely decline; this kit is for earned editorial only. |
| Silence after 60 days | Move on to alternative outlets (CNET, Digital Camera World, T3) using the same kit. |
